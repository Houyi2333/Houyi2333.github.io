首先先介绍一下 vscode 是个什么东西

有人会说 vscode 是宇宙第一 IDE，但是这句话是有问题的，因为 vscode 严格来说不算一个 IDE，只是一个文本编辑器。为什么他能编译运行呢，这要得益于他强大的插件库，是插件让一个文本编辑器变成了一个全能 IDE

然后，既然要用 vscode，肯定要安装 vscode

让我们打开 vscode 官网：Visual Studio Code - Code Editing. Redefined

点击红框里的 Download for Windows，之后浏览器就会自动下载

选择我同意此协议

五个复选框全部都打勾

点击安装

这样我们就能成功打开 vscode

但是现在是英文的，我们可以通过安装中文插件将其汉化

重启后我们就得到了中文 vscode

我相信大家的电脑上都装了 devc++，或装了 MinGW，这时候检查一下环境变量的 Path 有没有 MinGW 的 bin 路径，如果没有就给他加上

我们右键 devc++的快捷方式，单击“打开文件所在的位置”

打开“MinGW64”这个文件夹

打开“bin”这个文件夹

这样我们就找到了路径

这时候我们再装两个插件，一个叫 C/C++，一个叫 Code Runner

然后我们在设置里搜索 code-runner.runInTerminal ，勾选这一项

之后我们点击这个按钮就能在内部终端运行.c 程序了

那如果你习惯了用外部终端，可以这样设置一下
