如何使用vscode编译运行.c and .cpp文件

2024年01月02日 留言

首先先介绍一下vscode是个什么东西

有人会说vscode是宇宙第一IDE，但是这句话是有问题的，因为vscode严格来说不算一个IDE，只是一个文本编辑器。为什么他能编译运行呢，这要得益于他强大的插件库，是插件让一个文本编辑器变成了一个全能IDE

然后，既然要用vscode，肯定要安装vscode

让我们打开vscode官网：Visual Studio Code - Code Editing. Redefined

点击红框里的Download for Windows，之后浏览器就会自动下载

选择我同意此协议

五个复选框全部都打勾

点击安装

这样我们就能成功打开vscode

但是现在是英文的，我们可以通过安装中文插件将其汉化

重启后我们就得到了中文vscode
